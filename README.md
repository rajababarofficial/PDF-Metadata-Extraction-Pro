# 📄 PDF Metadata Extraction Pro

A high-performance desktop utility built to extract, analyze, and structure deep metadata from multiple PDF files simultaneously. Designed for digitization professionals, archivists, researchers, and data managers who require speed, accuracy, and structured export compatibility.

---

## ✨ Core Features

### 🖥 Modern GUI

Built with `CustomTkinter` for a clean, responsive, and professional desktop experience.

### 🧠 Intelligent Metadata Structuring

Automatically categorizes extracted data into a logical hierarchy:

1. **Primary Information**

   * File Name
   * Title
   * Author
   * Page Count

2. **Custom Metadata**

   * Detects and extracts unique user-defined metadata fields
   * Automatically organizes dynamic keys

3. **System Properties**

   * File Size (MB)
   * Producer
   * Creator
   * Creation & Modification Timestamps

---

### ⚡ Batch Processing

* Scan entire folders instantly
* Process hundreds of PDFs in seconds
* Optimized for digitization workflows

### 📊 Excel-Optimized Export

* Generates `UTF-8-SIG` encoded CSV
* Fully compatible with Microsoft Excel
* Clean column structure
* No encoding issues for Urdu / Sindhi / multilingual PDFs

## 🚀 Quick Start

### 📌 Prerequisites

* Python 3.10+
* `PyPDF2`
* `CustomTkinter`

---

### 🔧 Installation

#### 1️⃣ Clone Repository

```bash
git clone https://github.com/rajababarofficial/PDF-Metadata-Extraction-Pro.git
cd PDF-Metadata-Extraction-Pro
```

#### 2️⃣ Install Dependencies

```bash
pip install customtkinter PyPDF2
```

#### 3️⃣ Run Application

```bash
python Script.py
```

---

## 🛠️ How to Use

1. Click **“Choose Source Folder”** to select your PDF directory
2. (Optional) Define a custom export filename and save location
3. Click **“RUN EXTRACTION”**
4. Your structured CSV report will be generated instantly
